# /explore [탐색 대상]

`explorer` 에이전트를 호출하여 코드베이스를 탐색합니다.

## 사용 예시

```
/explore 코스 생성 API가 어떻게 동작하는지
/explore ChatRoom 컴포넌트 관련 파일
/explore 인증 토큰이 어떻게 전달되는지
/explore 동행 게시판 관련 모든 파일
```

## 실행 방식

`explorer` 에이전트에 위임:
> "[탐색 대상]을 TripMate-Jeju 코드베이스에서 찾아서 흐름과 관련 파일을 정리해줘"

에이전트가 메인 컨텍스트를 오염시키지 않고 탐색 후 결과만 반환합니다.

## 인자 없이 실행 시

전체 프로젝트 구조 요약을 반환합니다:
- 프론트엔드 주요 페이지 목록
- 백엔드 API 라우터 목록
- 최근 변경된 파일
